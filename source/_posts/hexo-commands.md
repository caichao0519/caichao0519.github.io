# 常用命令

`hexo n "my-first-blog" == hexo my-first-blog "我的博客" #新建文章`

`hexo g == hexo generate #生成`静态页面
`hexo s == hexo server  #以静态模式启动页面预览 http://localhost:4000`
`hexo d == hexo deploy #部署`到github pages



`hexo p == hexo publish`
